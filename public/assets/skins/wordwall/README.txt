WORDWALL SKIN SLOT
==================
Drop the real Wordwall "Open the Box" assets here, using these exact filenames:
  - wood.png        background (any size, will be scaled to cover)
  - box-card.png    the box/notebook front. Use a WHITE/light card so it can be
                    tinted to each question's color. Transparent outside the card.
  - box-spiral.png  the spiral binding strip drawn over the top of the card
                    (NOT tinted). Transparent background.

Then set SKIN = "wordwall" in src/config/skin.ts to use them.
The generated default skin lives in ../notebook with the same filenames.
